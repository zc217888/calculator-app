# 在线计算器网站

## Core Features

- 基础数学运算

- 科学计算功能

- 计算历史记录

- 响应式设计

- 键盘快捷键支持

## Design

现代简约的Material Design风格，深蓝色主色调，包含计算显示区域、按钮操作区和历史记录侧边栏，支持基础和科学计算模式切换
